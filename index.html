Understanding Optional Chaining in JavaScript

Posted on June 29, 2025

Optional chaining (`?.`) is a powerful feature in JavaScript that allows us to safely access deeply nested properties without having to check if each level exists. 


const user = {
  profile: {
    name: "Andy",
  },
};

console.log(user?.profile?.name); // "Andy"
console.log(user?.settings?.theme); // undefined (no error)
